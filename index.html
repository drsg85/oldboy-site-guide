Общие требования к заполнению

  1. На сайте для заполнения используются исключительно кавычки–ёлочки « » (если это не оговорено отдельно)

    Эти символы можно либо скопировать и вставить, либо использовать набор с помощью клавиши Alt и цифрового кода символа, набранного на цифровой клавиатуре.

    « = Alt + 0171
    » = Alt + 0187

  2. На сайте, в качестве знака препинания, НЕ используется знак минус («дефис») -. Дефис это не пунктуационный знак, а орфографический и используется только внутри слов.

    Для добавления знаков «короткое тире» и «длинное тире», следует использовать символы – и —. Можно копировать или вставлять с помощью Alt–кодов

    – = Alt + 0150
    — = Alt + 0151

  3. При создании и редактировании текста, мелкие части речи (такие, как союзы и предлоги) не должны оставаться висеть в конце строки.

    Для переноса таких частей текста, необходимо связать их с последующим словом при помощи «неразрывного пробела». Вставляется неразрывный пробел с помощью кода &nbsp;. Этот код нужно поставить между связываемыми словами ВМЕСТО пробела.

    Неразрывный пробел так же можно использовать и в других ситуациях, когда нужно как бы склеить два слова, чтобы они переносились вместе.

  4. В противоположной ситуации, когда необходимо наоборот, сделать принудительный перенос на новую строку, следует использовать тэг <br>

    При использовании тэга <br>, не стоит забывать про пробелы. Ставим перед или после <br>, в зависимости от ситуации (надо проверять как выглядит вживую на сайте). Это важно для SEO.

    Неправильно: слово<br>слово. Правильно: слово <br>слово, слово<br> слово

  5. Контентные изображения следует готовить следующим образом:

    • Изображение кадрируются по необходимым размерам (обычно они указываются в соответствующем пункте данного руководства).
    • Затем необходимо сохранить изображение в формате JPEG, со 100% качеством. В фотошопе это делается следующим образом: нажимаем Ctrl+Shift+Alt+S, в появившемся окне выбираем формат JPEG и качество 100% и сохраняем файл.
    • Сохраненные изображения затем необходимо пережать с помощью сервиса TinyPNG.

Заполнение филиалов

  • Поле «Адрес кратко»

    Содержимое этого поля отображается только в админке. Здесь следует записать как можно более полный адрес филиала, для того, чтобы можно было быстро и удобно искать нужный филиал через поиск.

    Формат: Город, поселок (п.), улица (ул.)/ проспект (пр-т.), дом (д.) ТЦ/ТРЦ.

  • Поле «Приложение Google Play»

    Данное поле заполняется системой автоматически. Следует изменять только в явных случаях.

  • Поле «Приложение Apple App Store»

    Данное поле заполняется системой автоматически. Следует изменять только в явных случаях.

  • Поле «Адрес»

    Содержимое этого поля отображается на странице филиала и в выпадающем списке филиалов

    Город НЕ указываем (отмечается в следующем поле).

    Формат: Если есть поселок (п.), улица (ул.) / проспект (пр-т.) / площадь (пл.), дом (д.), корпус (к.).

    ТЦ/ТРЦ, этаж, «во дворе» — УКАЗЫВАЕМ В ПОЛЕ «ОРИЕНТИР». Исключениями являются филиалы, где фаранчайзи сами обозначают формат адреса (полностью писать дом, корп., сведения о ТЦ, этаж).

  • Поля «Город» и «Город (текст)»

    Отображается на странице под строкой «Выбери свой город» и под логотипом «OLDBOY BARBERSHOP» на странице филиала.

    Заполняется одно поле из двух. Выбираем какое поле заполнять по следующему принципу:

    Если в городе только один филиал — пишем название города в поле «Город (текст)».

    Если в городе несколько филиалов, но необходимо сначала занести город в админку (создать страницу города). А затем начать вводить название города в поле «Город» и выбрать нужное значение из предложенных.

  • Поле «Телефон»

    Единый формат мобильных номеров для России: +7 012 345-67-89.

    Единый формат городских номеров для России: +7 (4012) 45-67-89.

    Исключениями являются филиалы, где фаранчайзи сами обозначают формат телефона (код города в скобках и др. замечания. Например: 7-8888-9).

  • Поле «Телефон (ссылка)»

    Данное поле нигде визуально не отображается, а является ссылкой для набора телефона из мобильной версии сайта.

    Формат: +70123456789. (Номер телефона без каких либо промежуточных, разделяющих символов).

    Пример: https://facebook.com/oldboy2015

  • Поле «Услуги»

    Отображается в виде таблицы с прайслистом на странице. Заполняется в формате HTML–разметки.

    Информацию для заполнения можно взять из yClients. Саму разметку можно скопировать из уже заполненых филиалов и изменить значения цен и услуг

    Примечания:

    • Если цена указана не в рублях, то перед стоимостью услуги, необходимо указать валюту, используемую в стране филиала (Украина — ГРН, Туркменистан — TMT, Европа — €, и т. д.)
    • Если стоимость услуги варьируется, то необходимо указывать диапазон цен через короткое тире. До и после тире ставим пробелы. Например: €10 – €100. Исключениями являются филиалы, где фаранчайзи сами обозначают формат прайса.
    • Если стоимость услуги не ограничена по верхней границе, то указываем только «от».

    Это поле необходимо для добавления на страницу произвольной информации (баннер, описание филиала, нестандартные счётчики и т. п.)

Как реализовать микроразметку на сайте Oldboy Barbershop

  • Сначала заполнить для каждого филиала следующий шаблон, а точнее - заполнить данные выделенные красным текстом.
  • После чего, каждому филиалу добавить на страницу (в админке) уже заполненный его шаблон микроразметки.
                    <script type="application/ld+json">
                        {
                        "@context": "http://schema.org",
                        "@type": "LocalBusiness",
                        "url": "URL ФИЛИАЛА (пример: https://oldboybarbershop.com/spb/ozerki-toreza)",
                        "name" : "OldBoy Barbershop ГОРОД",
                        "brand" : "OldBoy",
                        "logo": "https://oldboybarbershop.com/sites/all/themes/oldboy8/dist/img/logo.svg",
                        "telephone": "ТЕЛЕФОН(пример формата: +7(962)699-69-91)",
                        "email" : "ЭЛЕКТРОННАЯ ПОЧТА",
                        "branchOf" : "OldBoy",
                        "address": 
                            {
                                    "@type" : "PostalAddress",
                                    "streetAddress" : "АДРЕС",
                                    "addressLocality" : "ГОРОД",
                                    "addressCountry" : "СТРАНА"
                            },
                        "openingHoursSpecification" :
                            {
                            "@type" : "OpeningHoursSpecification",
                                "dayOfWeek": 
                                {
                            "@type" : "DayOfWeek",
                                "name" : "ПН-ВС"
                                    },
                            "opens" : "00:00",
                            "closes" : "00:00"
                            }
                        }
                    </script>
